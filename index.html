Crystals Collector

Instructions:


  1. You will be given a random crystal value to match at the start of the game.
  2. There are 4 crystals below. To reveal each crystal's value, you must click on it.
  3. Each time you click on a crystal, you will add that crystal's value to your score.
  4. You win the game by matching your crystal value score to the computer generated crystal value.
  5. You lose the game if your crystal value score exceeds the computer generated crystal value.
  6. Each time the game restarts, the value of the 4 crystals will change.
